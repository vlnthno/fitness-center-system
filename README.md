# ASID Fitness Center System

A simple console-based Gym Management System written in C++ to manage gym members, workouts, usage records, and payments.

## Features

* **Member Module:** Add new members, search for existing members, and renew memberships.
* **Workout Module:** Allocate body-part workout plans and log daily gym facility usage.
* **Payment Module:** Record customer payments with automated fee status checking (Paid, Underpaid, Overpaid).
* **Reports:** Generate system-wide summaries for admins or individual breakdowns for customers.
